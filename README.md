# foundation ES6 AMD TDD
Just a place where i can experiment with architectural approaches for building a framework with TDD and ES6

### Install
	npm install
	npm install -g grunt-cli

### Usage
	grunt

### Testing
    npm install -g testem
    npm install -g phantomjs 
    testem
